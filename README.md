Go 自从 1.11 版本以来就包含了对版本化模块(versioned modules)的支持（[链接](https://golang.org/design/24301-versioned-go)）。最初的原型`vgo`是在2018年2月发布的，2018年7月，版本化模块正式登陆Go的主分支。
